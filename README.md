Erstes Semester - Programmierprojekte

Sammlung von Praktikumsprojekten aus dem 1. Semester (PG1), entwickelt in C++ im Team.

Minesweeper
Terminal-basiertes Minesweeper mit zufälliger Minenverteilung und Aufdecken von Feldern

Schiffe versenken
Klassisches "Schiffe versenken" im Terminal mit Spielfeld-Verwaltung und Treffer-/Fehlschuss-Logik.

Dino-Herden-Simulation
Simulation einer Dinosaurier-Herde mit verschiedenen Verhaltensmustern und Interaktionen zwischen den Individuen.
